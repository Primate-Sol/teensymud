Class World
In: tmud.rb
Parent: Object

The World class is the mother of all worlds.

It contains the database and all manner of utility functions. It’s a big global thing.

db
is a handle to the database.
cmds
is a handle to the commands table (a ternary trie).
tits
is a handle to the tits event queue (an array).
options
is a handle to the configuration options structure.
