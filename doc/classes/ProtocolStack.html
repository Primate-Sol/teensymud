Class ProtocolStack
In: lib/protocol/protocolstack.rb
Parent: Object

The ProtocolStack class implements a stack of input and output filters. It also maintains some interesting state variables that are shared amongst filters. It keeps its own log.

Remarks:This should have its own configuration file.
